Performance & System Health

Scope

A lightweight native macOS menu bar utility for quick computer metrics with a storage-first point of view. See exactly what your machine is doing — CPU, memory, disk, battery, network — without opening Activity Monitor.

View on GitHub ← All Tools
What it does

Your Mac, made legible.

A quiet menu bar presence with a detailed view when you need it.

Scope lives in your menu bar as a small icon with visual load bars — no numbers cluttering your toolbar. Click it and a compact dashboard opens with everything that matters: battery and power state, top estimated app draw, CPU and GPU breakdown, memory pressure, network activity, and storage state.

The detail window goes deeper: storage cleanup candidates, power-impact apps, high-memory processes, active network destinations, battery health, thermal state, and a full system status overview. Everything is explained plainly — no jargon, no dark patterns pushing you toward paid upgrades.

Platforms macOS — built with Swift. Run with swift run Scope or build a local app bundle with the included script.
