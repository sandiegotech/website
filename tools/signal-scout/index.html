File Intelligence

Signal Scout

A local file triage tool for macOS. Choose a folder, and Signal Scout builds a read-only inventory that highlights what's useful, what's noise, what's generated, and what's duplicated. It never moves, renames, or rewrites anything.

View on GitHub ← All Tools
What it does

Find the signal in your file system.

Read-only triage. Nothing changes unless you decide it should.

Large project folders accumulate noise: generated output, duplicate assets, stale caches, and files that were useful once. Signal Scout scans the folder you choose and produces a structured inventory that breaks files into categories — useful, noisy, generated, duplicated, or worth reviewing — with visible scan progress and smart pruning of generated-folder subtrees.

The first version is intentionally careful. It shows you what it found and why, then gets out of the way. The decision of what to do belongs to you.
